# Dashboard Indicadores

Financial indicators dashboard built with **React**, **TypeScript**, **Vite** and **SASS**.

This project was created as a practical frontend exercise focused on building a clean, maintainable and scalable dashboard using modern React practices.

## 🚀 Features

- Financial indicators dashboard UI.
- React + TypeScript implementation.
- Vite-based development environment.
- SASS styling.
- Component-based structure.
- Clean code practices.
- Focus on maintainability and frontend architecture.

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- SASS
- ESLint

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/roberto211093/dashboard-indicadores.git
cd dashboard-indicadores
```

Install dependencies:

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Build for production:

```bash
pnpm build
```

Preview production build:

```bash
pnpm preview
```

## 📂 Project Purpose

The goal of this project is to practice and demonstrate frontend development skills using React and TypeScript, with attention to UI organization, reusable components, styling strategy and maintainable project structure.
